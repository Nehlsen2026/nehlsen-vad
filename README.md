# Nehlsen MV – Abfragebögen Außendienst

## Dateien
| Datei | Zweck |
|---|---|
| `index.html` | Die App (alle 8 Branchen-Bögen) |
| `manifest.json` | PWA-Konfiguration (App-Name, Icon, Farbe) |
| `sw.js` | Service Worker (Offline-Funktion) |
| `icon-192.png` | App-Icon für Android |
| `icon-512.png` | App-Icon für Splash Screen |

## Auf GitHub Pages hochladen

1. GitHub-Account erstellen (kostenlos): https://github.com
2. Neues Repository anlegen: **"nehlsen-vad"** (öffentlich)
3. Alle 5 Dateien hochladen (Drag & Drop)
4. Einstellungen → Pages → Branch: main → Speichern
5. URL: **https://[IHR-NAME].github.io/nehlsen-vad/**

## VAD-Anleitung (Handy)

### Android
1. Link im Chrome öffnen
2. Oben rechts: ⋮ → "Zum Startbildschirm hinzufügen"
3. ODER: Banner "App installieren" antippen

### iPhone (iOS)
1. Link im Safari öffnen
2. Unten: Teilen-Symbol → "Zum Home-Bildschirm"
3. Name bestätigen → Hinzufügen

## PDF pro Kunde
1. Bogen ausfüllen (Kundenname in Stammdaten eintragen)
2. "PDF speichern" tippen
3. Dateiname wird automatisch auf Kundenname gesetzt
4. "Als PDF speichern" wählen
